Arrays

Important Points:
  • One Dimensional Arrays can be declared in following ways: int[]a, int []a, int a[], int[]a. Similarly, Two Dimensional Arrays can be declared in following ways: int [][]a, int[] []a, int a[][], int[] a[]. But int[][] a,[]b will give compile time error.
  • int[] a = new int[-10]; will throw NegativeArraySizeException at runtime.
  • Array creation with both dimension expression and initialization is illegal eg. int[] a = new int[4]{100,200,300,400};. The correct way is int[] a = new int[]{100,200,300,400};.
  • One approach to initialize 2D arrays is int[][] a={{100,200,300},{400,500,600},{700,800,900}}. Now If you do System.out.println(a); Result will be [[I@someHashValue where '[[' signifies 2d array and 'I' stands for int datatype.
  • Declaring a 2-D Jagged array. Arrays with no fixed number of columns. 
                            
        int arr[][] = new int[2][];
 
        // First row has 3 columns
        arr[0] = new int[3];
 
        // Second row has 2 columns
        arr[1] = new int[2];

 class Solution {
    public int[] twoSum(int[] nums, int target) {
        int[] a = new int[]{-1,-1};
        for(int i=0;i<nums.length-1;i++){
            for(int j=i+1;j<nums.length;j++){
                if(nums[i]+nums[j]==target){
                    a[0]=i;
                    a[1]=j;
                    break;
                }
            }
        }
        return a;
    }
}
