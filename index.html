DSA Practice in JAVA

Arrays

Important Points:
  • One Dimensional Arrays can be declared in following ways: int[]a, int []a, int a[], int[]a. Similarly, Two Dimensional Arrays can be declared in following ways: int [][]a, int[] []a, int a[][], int[] a[].
  • int[] a = new int[-10]; will throw NegativeArraySizeException at runtime.
  • Array creation with both dimension expression and initialization is illegal int[] a = new int[4]{100,200,300,400};. The correct way is int[] a = new int[]{100,200,300,400};.
